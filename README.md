# 🛒 Blinkit Sales Dashboard

## 👋 About this project

I built this Power BI dashboard to explore and understand Blinkit's grocery sales data in a simple and visual way.
The goal was to turn raw data into something meaningful and easy to interpret.

---

## 📊 What’s inside

This dashboard helps answer questions like:

* How are overall sales performing?
* Which products are selling the most?
* Do different outlet sizes or locations affect sales?
* How do customer ratings look?

---

## 🔍 Key highlights

* Total Sales: $1.20M
* Avg Sales: $141
* 8500+ items analyzed
* Avg Rating: 3.9

---

## 🛠️ Tools I used

* Power BI (for building the dashboard)
* Excel (for the dataset)

---

## 🚀 How to explore

Just open the `.pbix` file in Power BI and play around with the filters —
you can slice the data by outlet type, size, and item category.
